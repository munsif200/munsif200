### Local Development
```bash
# Clone the repository
git clone https://github.com/munsif200/munsif200.git
cd munsif200

# Install dependencies (requires Ruby and Jekyll)
bundle install

# Serve locally
bundle exec jekyll serve

# View at http://localhost:4000
```
